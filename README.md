# Les Os du Destin

Jeu de taverne pour le serveur JDR dark fantasy Luxcyros.

**Jouer en ligne :** https://arugas-afk.github.io/Jeu-de-taverne-Les-Os-du-destin/os-du-destin.html

## Règles rapides

- Trois joueurs ou plus lancent chacun 3D6.
- Seules les sommes **paires** comptent ; les sommes impaires éliminent leur joueur.
- La somme paire la plus haute l'emporte.
- Égalité de somme : celui qui a le plus de "6" gagne.
- Égalité parfaite (même somme, même nombre de 6) : nouvelle manche entre les joueurs à égalité.
